DATABASE STRUCTURES

by - Name - Rajveer Kaur Dhaliwal | Student ID - 202105291

Database Structures refer to how information is organized and stored in the database of the Warehouse Management System. It outlines the tables, fields, and relationships between data, which helps in managing and accessing information effectively within the system.

Inventory Table

  • Fields: Item ID, Name, Description, Quantity, Location, Supplier ID, Cost, etc.
  • Relationships: Connected to the Supplier table for supplier details.

    • Order Table

  • Fields: Order ID, Customer ID, Date, Status, Total Amount, etc.
  • Relationships: Linked to the Inventory table for items included in the order.

    • Supplier Table

  • Fields: Supplier ID, Name, Contact Information, Product Catalog, etc.
  • Relationships: Associated with the Inventory table for products supplied.
